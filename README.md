## Описание (что изучили)

В ходе выполнения лабораторной работы были изучены следующие темы:

### 1. Основы JavaScript
- Место JavaScript в web-разработке
- Способы подключения JavaScript к HTML (в тег `body`)
- Запуск JavaScript-кода в браузере и через Node.js

### 2. Циклы в JavaScript
- Цикл `for` — вывод чисел от 1 до 10, чётные числа, подсчёт суммы
- Цикл `while` — уменьшение значения переменной до 0
- Цикл `do...while` — гарантированное выполнение хотя бы одной итерации
- Операторы `break` и `continue` для управления выполнением циклов
- Вложенные циклы для вывода фигур

### 3. Функции в JavaScript
- Объявление функций (классический способ)
- Функции без `return`
- Параметры по умолчанию
- Функции как значения (аналог делегатов в C#)
- Замыкания (closures)
- Стрелочные функции (arrow functions) — современный синтаксис

### 4. Массивы в JavaScript
- Создание массивов (разные способы)
- Основные методы: `push()`, `pop()`, `shift()`, `unshift()`
- Перебор массива: `for`, `for...of`, `forEach()`
- Поиск элементов: `indexOf()`, `includes()`
- Массивы с элементами разных типов данных

### 5. Объекты в JavaScript
- Создание объектов (object literal) без класса
- Доступ к свойствам (через точку и квадратные скобки)
- Добавление и удаление свойств на лету
- Объекты с методами
- Перебор свойств объекта (`for...in`)
- Вложенные объекты и массивы
